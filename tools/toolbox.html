Purpleprint Toolbox (UE4)



Get Purpleprint Toolbox $19.99 + VAT

Introduction


Is an easy to use package of tools designed to speedup the process and the performance in your games. Purpleprint - Toolbox provide suite of tools for (Unreal Engine 4 Only) that can convert world Static Meshes to Blueprint Instance Meshes or Instance Meshes to Static Meshes. (The features and actors only work using the provided actors, the meshes from the screenshots aren't included).




Static Instances Manager

Selected static meshes from the world and generate a static mesh instance actor!

Generate Instances In One Click

Generate an instance from the viewport selection

Edit Instances

Edit the instances locations, rotations, scales

Combine Instances

Join multiple actor instances with the same mesh into the same actor


Array Instance Actor

Generate an array of meshes in 1D, 2D and 3D space

Total control over the instance meshes

Different Settings

Setup the instances as you want

Random Settings

Rotate, scale, move and align the meshes


Spline Instance Actor

Generate an array of meshes following a spline in 3D space

Total control over the instance meshes

Different Settings

Setup the instances as you want

Random Settings

Area Instance Actor

Details about Purpleprint Toolbox (UE4)

Description


Name: Purpleprint Toolbox

Platforms: PC Unreal Engine 4.18+ Unreal Engine 4.24+ Itch.io

Developer: Hevedy

Publisher: Hevedy

Current Version: 1.2.4

Release Date: 2018/01/12 (Y/M/D)


Minimum Requirements


Any computer that works with Unreal Engine 4.18 or Unreal Engine 4.24

Space: At least 100 MB of free space



Features


> Create static instances from selected static meshes.

> Procedural arrays and splines of instances.

> Select world meshes filtered by tag or mesh.

> Select meshes close to selection aswell filtered.

> Combine static actor instances into a single one.

> Add selected meshes to a static instance.

> Create static meshes from selected static instances actor.

> World align X,Y,Z / X-,Y-,Z-.

> Random locations, rotations and scales.

> Random textures with same material example.
