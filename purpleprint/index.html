Core

Frequently Asked Questions

  • Whats this tool for?

    Purpleprint is an easy to use package of plugins with tools designed to speedup the process and the performance in your games.

  • Does the plugins depends between them?

    Only the Purpleprint - Core plugin is needed for them all, the rest work individually using Core and engine included plugins.

  • Does the licenses allows me to use it for commercial works?

    The only official stores are FAB, Itch.io or Gumroad, both licenses Individual/Personal and Studio/Professional allow the comercial use of the plugins under a DRM-Free, royalty-free, platform-independent, non-expiring license. The license is per seat, if you are a team or studio with more than $100k of revenue or funding in the last 12 months, then need the "Studio/Professional License", otherwise use the "Individual/Personal License".

  • Are the versions on the different marketplaces different?

    No, it just depends of the moment they get updated, ones take more to be updated than others, but versions should be the same.

  • Need a super computer to run this? (Minimum Specs)

    Actually the specs are ruled by the version of Unreal Engine you are using, you can check the minimum specs list. hardware-and-software-specifications-for-unreal-engine
    *ONLY works under DirectX (11 or 12) (SM5 or SM6) on Windows +10 (x64)
